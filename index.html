Мощный декомпилятор JAR файлов

CFR Decompiler Kit - это удобная оболочка для CFR, современного декомпилятора Java, который восстанавливает исходный код из скомпилированных .jar файлов с высокой точностью. Поддерживает функции Java 8+, включая лямбды, switch-выражения и многое другое.

Поддержка Java 8+
Простой .bat скрипт
Открытый исходный код
DecompilerDemo.java
// Декомпилировано с помощью CFR 0.152
import java.util.function.Function;

public class DecompilerDemo {
    public static void main(String[] args) {
        Function<String, String> toUpper = String::toUpperCase;
        System.out.println(toUpper.apply("cfr decompiler"));
    }
}

Основные возможности

Поддержка Java 8+

Работает с современными функциями Java: лямбды, ссылки на методы, switch-выражения и другие.

Высокая производительность

Оптимизированный движок декомпиляции, эффективно работающий даже с большими .jar файлами.

Читаемый вывод

Генерирует чистый, читаемый Java-код, сохраняя оригинальную структуру где это возможно.

Удобный .bat скрипт

Простой в использовании .bat файл для быстрой декомпиляции .jar файлов.

Без зависимостей

Один JAR-файл без внешних зависимостей, легко интегрируется в ваш рабочий процесс.

Обработка ошибок

.bat файл в комплекте умеет распозновать ошибки

Как установить CFR Decompiler Kit

1

Распакуйте архив

Распакуйте архив в папку C:\CFR\. Архив содержит файлы cfr-0.152.jar и decompile.bat. 

C:\CFR\
├── cfr-0.152.jar
└── decompile.bat
2

Добавьте в Path

Добавьте C:\CFR\ в системную переменную Path, чтобы можно было вызывать декомпилятор из любой папки. 

Path=C:\Windows\...;C:\CFR\
3

Используйте

Откройте консоль в любой папке и используйте команды:

decompile [jar]
decompile [jar] [output folder]
